# Onboarding Plan

This is the intended flow for a future public user.

## Phase 1

Install the distro:
- clone repo
- install dependencies
- choose model providers
- run setup wizard

## Phase 2

Configure private values:
- Tailscale
- Telegram bot token
- owner chat id
- optional OAuth providers
- optional local model endpoints

## Phase 3

Enable default agent pack:
- DOC
- CODEX
- PA
- HK

## Phase 4

Enable optional modules:
- FINANCE overlay
- LEGAL overlay
- GitNexus adapter

## Phase 5

Run validation:
- gateway config validation
- memory lane smoke
- Telegram delivery smoke
- local model healthcheck
- optional repo-intelligence smoke
