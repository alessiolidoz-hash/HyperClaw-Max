# GitNexus Adapter

GitNexus is a candidate second engine for repo-centric intelligence.

Current public-distro stance:
- optional adapter
- not mandatory core

Why:
- technically strong for repository indexing and code intelligence
- currently not ideal as bundled OSS heart because the local package metadata declares a noncommercial license

What this adapter should eventually provide:
- repo indexing setup
- compare queries
- optional upstream and donor lookup
- adapter boundary so a future OSS replacement can slot in
